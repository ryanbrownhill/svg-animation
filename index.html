SMIL & CSS

SMIL?

SMIL means Synchronized Multimedia Integration Language, which is an XLM-based language

Why SMIL?

Can animate attributes that CSS can't

Works in all browsers except IE and Opera Mini

SMIL Animation Tags

<animate/> For animating attributes & properties <animateTransform/> For animating transform properties <animateMotion/> For animating motion on path (hopefully coming soon to css)
